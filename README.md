# React.js-MonsterHunterWorldIceborne-Quest-Management-Website-Template-Korean

제작일: 2025-12월-20일 19시

React.js를 기반으로  Monster Hunter World: Iceborne™ 게임을 배경으로 한 퀘스트 관리 웹사이트 프로젝트입니다.

가천대학교 글로벌 캠퍼스, 컴튜터공학부 2학년 2학기 때 개발한 학기말 프로젝트입니다.

실행 방법/How to run:
1. 폴더 압축 풀기
   Extract ZIP File
2. cmd 터미널에서 부모 디렉터리에 접속하여 "npm run dev" 실행.
   run "npm run" in Parent Directory in cmd Terminal
3. cmd 터미널에 "Local URL: http://localhost:****/"가 출력될 것입니다. 이를 브라우저에 입력해서 접속하시면 됩니다.
   In the cmd Terminal, the URL of the app will be shown. You can access the app via copying the URL and pasting it on your browser
